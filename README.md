### 本地运行（完整功能）

**要求：** Node.js v18+（没有？去 [nodejs.org](https://nodejs.org) 下载安装）

**步骤：**
1. 下载并解压本项目
2. 双击项目里的 `run.bat`
3. 等待自动安装依赖 → 自动打开浏览器

> 首次运行会自动安装依赖（约1-2分钟），之后每次直接双击 `run.bat` 即可。

## 常见问题

**Q: 双击 run.bat 没反应？**
→ 确保已安装 [Node.js](https://nodejs.org)，安装后重新双击 run.bat

**Q: 提示端口被占用？**
→ run.bat 会自动关闭占用 5173 和 3003 的程序；或手动关闭相关窗口

**Q: 收不到验证码？**
→ 本地运行验证码固定为 `123456`（测试模式）
